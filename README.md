# Data-Visualisation_TATA_Forage

## 📌 Overview
This project was completed as part of Tata’s Data Visualisation job simulation on the Forage platform. The objective was to analyze retail revenue data and create visualizations to support business decision-making.

---

## 📊 What I Did
- Analyzed retail revenue data  
- Created visualizations using Tableau  
- Explored trends and patterns in the data  
- Presented insights in a clear and structured way  

---

## 🛠 Tools Used
- Tableau

---

## 📊 Visual Insights

### 1. Revenue Trend 
Shows monthly revenue trends for 2011, helping identify seasonal patterns.

### 2. Top 10 Countries by Revenue 
Displays the highest revenue-generating countries excluding the United Kingdom.

### 3. Top 10 Customers by Revenue
Highlights top customers ranked by revenue contribution.

### 4. Demand by Region
Shows countries with the highest demand, useful for expansion decisions.
